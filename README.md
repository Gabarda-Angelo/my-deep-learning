# PyTorch Deep Learning Projects

## Overview

This repository documents hands-on work focused on building, training,
optimizing, and deploying deep learning models using the PyTorch
framework. It follows a structured progression from foundational
concepts to advanced architectures and real-world deployment workflows.

The goal of this project is to develop practical skills in designing
neural networks, applying them to real datasets, and preparing trained
models for production environments.

------------------------------------------------------------------------

## Learning Progression

### 1. Foundations of PyTorch and Neural Networks

-   Understanding tensors and tensor operations\
-   Building neural networks using `torch.nn`\
-   Implementing forward and backward propagation\
-   Creating machine learning pipelines\
-   Training loops, loss functions, and optimization\
-   Model evaluation and validation techniques

These fundamentals form the backbone of deep learning systems.

------------------------------------------------------------------------

### 2. Applied Deep Learning: Computer Vision and NLP

-   Working with image datasets using TorchVision\
-   Implementing and improving CNN architectures\
-   Using pre-trained models for transfer learning\
-   Fine-tuning models for custom tasks\
-   Natural Language Processing using transformer-based models\
-   Leveraging Hugging Face libraries for text tasks

Projects include: - Image classification\
- Text classification\
- Image segmentation

This phase focuses on applying theory to real-world tasks.

------------------------------------------------------------------------

### 3. Advanced Architectures and Model Optimization

Explored advanced neural network architectures such as:

-   Siamese Networks\
-   ResNet\
-   DenseNet\
-   Transformers

Model optimization techniques include:

-   Hyperparameter tuning\
-   Pruning\
-   Quantization\
-   Exporting models with ONNX\
-   Experiment tracking and management using MLflow

This stage emphasizes performance improvement and deployment readiness.

------------------------------------------------------------------------

## Applied Learning Projects

Throughout the project, models were:
